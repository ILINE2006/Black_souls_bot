🛡️ Автоматическая модерация (работает сама)
Бот постоянно следит за чатом без твоего участия:

Фильтр ссылок (Анти-спам): Автоматически удаляет любые сообщения, содержащие http://, https://, www. или discord.gg/.

Умный пропуск: Фильтр игнорирует сообщения от администраторов и модераторов (у которых есть право "Управлять сообщениями").

Предупреждения: При удалении ссылки бот отправляет нарушителю временное сообщение-предупреждение, которое само исчезает через 5 секунд, чтобы не засорять чат.

⚔️ Команды модерации (только для администрации)
Эти команды сработают только у тех пользователей, у которых есть соответствующие права на сервере:

/clear [количество] — массово удаляет указанное количество сообщений в чате (удобно для быстрой очистки спама).

/mute [пользователь] [минуты] [причина] — отправляет нарушителя в "тайм-аут". Пользователь остается на сервере, но временно не может писать в чаты и говорить в голосовых каналах.

/unmute [пользователь] — досрочно снимает наказание (тайм-аут) с участника.

/kick [пользователь] [причина] — выгоняет участника с сервера (он сможет вернуться, если снова получит ссылку-приглашение).

/ban [пользователь] [причина] — выдает перманентную блокировку. Пользователь не сможет вернуться на сервер ни при каких условиях, пока бан не будет снят вручную в настройках сервера.

ℹ️ Информационные команды (доступны всем)
Эти команды может использовать любой участник сервера:

/userinfo [пользователь] — выводит красивую карточку (Embed) с информацией об участнике. В ней отображается: аватарка, ID, упоминание, высшая роль, а также точные даты создания аккаунта в Discord и захода на твой сервер.

/help — показывает удобный справочник со всеми доступными командами. Это сообщение отправляется в скрытом режиме (его видит только тот, кто вызвал команду), чтобы чат оставался чистым.
